```bash
# Entenda o briefing
- Objetivo da experiência (landing, app, portfolio, ecommerce)
- Target audience & psicologia
- Brand personality & valores
- Diferencial vs concorrência
- KPIs de sucesso (conversão, engagement, viral)
```

### 2. **IA Brand System Generation**
```bash
# Use IA para gerar sistema de design
- Análise emocional do público-alvo
- Paleta científica baseada em psicologia das cores
- Tipografia que comunica personalidade
- Tokens de espaçamento harmônicos
- Motion design & timing curves
```

### 3. **Arquitetura 3D Impossível**

### 6. **Desenvolvimento Anti-AI Slop**

**❌ EVITAR (Padrões genéricos de IA):**
- Gradientes azul-roxo óbvios
- Tipografia system fonts (Inter, Roboto)
- Cards brancos com sombra sutil
- Animações bounce genéricas
- CTAs "Get Started" sem personalidade

**✅ FAZER (Design distintivo):**
- Paleta emocional específica para o brand
- Tipografia display única + body refinado
- Layouts que quebram o grid
- Animações cinematográficas (stagger, parallax)
- CTAs que falam a linguagem do usuário

### 7. **Mobile-First Extremo**
```bash
# Otimização móvel científica
- Touch-first design (44px+ touch targets)
- Gestos nativos (pinch, swipe, long-press)
- Performance sub-100ms
- Offline-first PWA
- WebXR para câmera/AR
- One-handed usability
```

### 8. **Performance Engineering**
```bash
# Lighthouse 100 garantido
- Critical CSS inlined
- Images WebP/AVIF + lazy loading
- 3D assets otimizados (LOD, compression)
- Bundle splitting inteligente
- Edge caching strategy
- Service Worker agressivo
```

### 9. **Asset Generation Pipeline**
```bash
# IA para assets únicos
- Icons: busca semântica em 10M+ ícones
- Images: Midjourney/Flux prompts otimizados
- 3D: Spline integration + Blender exports
- Videos: Remotion motion graphics
- Fonts: Custom typography pairing
```

### 10. **Deployment & Monitoring**
