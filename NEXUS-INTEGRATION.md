## 🔧 Setup & Installation

### **1. Integrar ao OpenClaw**
```bash
# Copiar skill para OpenClaw
cp /root/.openclaw/workspace/nexus-project/SKILL.md /usr/lib/node_modules/openclaw/skills/nexus/SKILL.md

# Registrar agents
ln -s /root/.openclaw/workspace/nexus-project/agents/* /usr/lib/node_modules/openclaw/skills/nexus/agents/

# Instalar dependências
npm install -g @nexus/cli three @react-three/fiber gsap
```

### **2. Configurar APIs**
```bash
# OpenAI para image generation
export OPENAI_API_KEY="sk-..."

# Anthropic para design intelligence  
export ANTHROPIC_API_KEY="sk-ant-..."

# Midjourney webhook (opcional)
export MIDJOURNEY_WEBHOOK_URL="https://..."
```

### **3. Validar Setup**
```bash
nexus --version
nexus validate-setup
nexus create hello-world --test
```
